Eclipse下配置Pydev开发环境

发表于 2016-05-11   |   分类于 教程   |     |   阅读次数

前言

在自己电脑上安装的是Anaconda Python环境，之前用Spyder IDE开发，这个IDE比较好用，但是最近不知何故，装了几个Python包以后突然挂了。自己又比较想在Eclipse里把Python，Perl，Scala等环境都配置好，所有选择在Eclipse下安装Pydev插件。

更新

  • 2016-05-11: 在window10/JDK8/Anaconda_Python2.7环境下安装Pydev

下载

Pydev 官方网站，在里面简单看下安装环境需求。

Release 5.0.0

  • Important PyDev now requires Java 8.
  • PyDev 4.5.5 is the last release supporting Java 7.

这里明确写了装最新版本的Pydev需要JDK8, 我采用的是Eclipse在线安装，会安装最新版本。

安装

  • 启动Eclipse, 点击Help->Install New Software…
  • 在弹出的对话框中，点Add 按钮。 Name中填:Pydev, Location中填http://pydev.org/updates

下一步：

然后一步一步装下去。如果装的过程中，报错了,请尝试直接从官网下载安装包，解压添加到eclipse里的方式。

配置

安装好pydev后,需要配置Python解释器

  • 在Eclipse菜单栏中,点击Windows->Preferences.
  • 在对话框中，点击pyDev->Interpreter-Python.
    • 点击New按钮,选择python.exe的路径,打开后显示出一个包含很多复选框的窗口. 点OK.
    • 或者直接点击Quick Auto-config按钮，我安装的是Anaconda测试时可以直接设置的。
